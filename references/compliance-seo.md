# 合规/SEO 参考

## 微信合规红线

### 绝对禁止
- 政治敏感内容（领导人、政策批评、政治事件）
- 色情/暴力/赌博
- 虚假信息/谣言
- 诱导分享/关注（"转发才能看"等）
- 抄袭/未授权转载
- 违规广告（医疗、金融等需资质领域）

### 低创检测机制
微信有低创内容检测，会识别：
- CSS 指纹完全相同的批量文章
- 高度模板化的排版
- 大段文字无图片/样式变化

### 对策
- **CSS 随机扰动**：converter.py 在排版时自动添加微小的 CSS 随机变化
  - 间距微调：±1-3px 随机偏移
  - 颜色微偏移：±1-3 随机色值偏移
  - 字号微调：±0.5px 随机偏移
- **排版多样性**：每篇文章使用不同的组件组合
- **AIGC 声明**：impeccable 主题自动追加 AI 辅助声明

## 图片标注规则

- AI 生成的封面图：标注"AI 生成封面"
- 信息图：标注"示意图，非真实数据"（如果数据为示意）
- 截图：标注来源

## SEO 注意事项

- 标题 20-28 字，含关键词
- 摘要 ≤120 字节
- 前 200 字内包含核心关键词
- 5 个标签：2 行业 + 2 热门 + 1 长尾
- 避免标题党（微信会降权）
