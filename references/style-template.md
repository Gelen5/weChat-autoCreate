# 风格配置字段 + 主题列表

## style.yaml 字段说明

```yaml
name: "我的公众号"          # 公众号名称
industry: "科技"            # 行业领域
topics: ["AI", "效率工具"]   # 内容方向
tone: "专业但亲切"           # 语气风格
writing_persona: "midnight-friend"  # 写作人格
blacklist: ["竞品A"]        # 竞品/敏感词
theme: "professional-clean" # 排版主题
cover_style: "impact"       # 封面风格 (impact/atmosphere/infographic)
author: "作者名"            # 作者名
content_style: "干货型"      # 内容类型 (干货型/测评型/观点型/故事型)
```
