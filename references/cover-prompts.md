
### 故事型文章


### 清单型文章
```
A clean infographic-style illustration representing [主题], organized geometric elements, data visualization aesthetic, [强调色] and neutral palette, no text, no letters
```

### 对比型文章

## 通用规则

1. 所有提示词必须包含 "no text, no letters, no words"
2. 必须包含至少 2 个具体实体
3. 必须指定配色方向
4. 不指定具体艺术家名字（避免风格争议）
