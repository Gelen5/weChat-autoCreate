# 选题评估规则

## 三维度评分

### 1. 热度（权重 30%）
- 微博热搜排名：top3 = 10分，top10 = 7分，top20 = 4分
- 百度热搜排名：同上
- 头条热榜排名：同上
- 多平台命中加分：2个平台 = +2，3个平台 = +4

### 2. 相关度（权重 40%）
- 与用户 topics 域匹配度：完全匹配 = 10分，部分匹配 = 6分，无关 = 2分
- 与 content_style 一致性：匹配 = +2
- 与历史选题重复度：重复 = -5

### 3. 切入价值（权重 30%）
- 信息增益：新视角/新数据/新结论 = 10分
- 新鲜度：24h内 = 10分，48h = 7分，72h = 4分，更久 = 2分
- 可操作性：有明确行动建议 = +3

## 综合评分公式

```
score = heat * 0.3 + relevance * 0.4 + insight * 0.3
```

## 历史去重规则

| 时间窗口 | 惩罚 |
|----------|------|
| 7天内写过相同选题 | -3分 |
| 30天内写过相同选题 | -1分 |
| 90天内写过相同选题 | 标注"旧题新角度" |

## 常青选题

当热点不足时，从用户 topics 域中选择常青选题：
- 通用方法论类（"如何高效学习X"）
- 框架/工具深度解读类
- 行业趋势分析类
- 个人经验复盘类

## 输出格式

```markdown
| # | 选题 | 热度 | 相关度 | 切入价值 | 总分 | 来源 |
|---|------|------|--------|---------|------|------|
| 1 | ...  | 8    | 9      | 7       | 8.1  | 微博+百度 |
```

推荐 10 个选题：7-8 个热点驱动 + 2-3 个常青。
